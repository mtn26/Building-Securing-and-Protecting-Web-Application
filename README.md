<h1>Securing Web Application</h1>

 

<h2>Description</h2>
This project consists of building, securing, and protecting a cloud application to host a cyber blog
<br />


<h2>Languages and Utilities Used</h2>

- <b>Bash/CloudShell</b> 
- <b>CSS</b>
- <b>SSH</b>

<h2>Environments Used </h2>

- <b>Google Chrome</b>
- <b>Azure</b>
- <b>Windows 10</b>
